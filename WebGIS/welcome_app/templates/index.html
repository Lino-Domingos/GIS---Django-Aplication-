trabalho desenvolvido permitiu a criação do SIG_Aquacultura - Sistema de informação geográfica baseado na web para Aquacultura. O SIG_Aquacultura é um gestor de informação geográfica cujo objectivo passa por centralizar e organizar dados geográficos relevantes para a gestão da Aquacultura, promovendo a sua disponibilização e difusão via internet, segundo níveis de acesso configuráveis e utilizando uma interface comum. A informação é disponibilizada através de tecnologia WebSIG, não havendo, portanto, qualquer custo associado à aquisição dos softwares utilizados. O trabalho desenvolvido permitiu demonstrar como é possível, de uma forma relativamente simples e sem grandes custos, disponibilizar informação geográfica e alfanumérica para um vasto público utilizando algumas das muitas aplicações Open Source disponíveis para descarga na internet. Foi assim possível construir uma ferramenta que considerá-se de muita utilidade para os demais intervenientes no sector da Aquacultura e que cumpre na sua totalidade os objectivos traçados inicialmente. Tendo por base as plataformas do Django (GeoDjango, Folium, OpenLayer, Leaflet, PostGIS entre outras), o SIG_Aquacultura foi dotado de um conjunto de ferramentas bastante completo.
